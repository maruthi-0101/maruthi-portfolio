**Mouse head tracking (critical feature):**
- On `mousemove`, normalize mouse position: `mouseX = (event.clientX / window.innerWidth) * 2 - 1` and `mouseY = -(event.clientY / window.innerHeight) * 2 + 1`
- In the `requestAnimationFrame` render loop, find the character's head/spine bone: `character.getObjectByName("spine006")` or `"Head"` or `"mixamorigHead"` (try all three, use whichever exists)
- Apply smooth rotation using lerp: `headBone.rotation.y = lerp(headBone.rotation.y, mouseX * (Math.PI/6), 0.1)` and `headBone.rotation.x = lerp(headBone.rotation.x, -mouseY * 0.3, 0.2)` — where `lerp(a,b,t) = a + (b-a)*t`
- **Max rotation clamped to** `Math.PI / 6` (30 degrees) on Y axis
